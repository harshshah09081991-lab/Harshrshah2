## Architecture

### Frontend Stack
- React 19.0.0
- React Router for navigation
- Shadcn UI components
- Tailwind CSS with custom animations
- Inter font (Google Fonts)
- Lucide React for icons
- Sonner for toast notifications

### Design System
- **Primary Colors**: Emerald (#059669, #10b981, #34d399)
- **Secondary Colors**: Cyan (#0ea5e9, #06b6d4, #22d3ee)
- **Dark Backgrounds**: Slate (#0f172a, #1e293b, #334155)
- **Typography**: Inter (300-900 weights)
- **Border Radius**: 0.5rem (base), up to 3rem for cards
- **Animations**: Float, glow, slide-in, shimmer
